# ElectricalPowerNetwork
Reference datasets for the Critical Point Multiplex Centrality Index (cpmci). This repository provides datasets in graph format (vertices, edges, and associated data) to support reproducible research, validation, performance evaluation, and the comparison of cpmci with other network centrality metrics.

For data reproducibility, please use the following address:
https://toni-arb.shinyapps.io/cpmciAPP/
